Doom II is definitely the least confusing release to sort out.  There
was only a single version 1.9, the most common release available.  It
is used here as the version every other one revolves around: you may
patch any version from or to 1.9.

Doom II 1.666g refers to the German release of the game, which removed
Nazi symbolism (id released an official patch to uncensor the game).
Doom II 1.8f is the French-translated version.

Doom II Zodiac is the version released for the Tapwave Zodiac hand
held console
